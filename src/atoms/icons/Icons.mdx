# Icons

UI-components offers a set of icons ready to be used in a web application.
Icons are built as React components using the `SvgIcon` component.

## Properties

Icons can be configured in the same way (have the same props) as the SvgIcon component (expect the `children` prop since the svg content is already set for each icon).
To see theses props in action, you can either go to the documentation of the SvgIcon component or open the
playground of the icons and add some props.

<Props of={PasswordsIcon} size={30} />
