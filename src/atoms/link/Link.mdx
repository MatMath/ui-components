## Usage

### Possible link config:

<Playground
  style={{
    display: 'flex',
    flexDirection: 'row',
    justifyContent: 'space-around'
  }}
>
  <Link>Need help?</Link>
  <Link disabled>Need help?</Link>
  <Link size='small'>Need help?</Link>
</Playground>

### The default theme is light (white), but a link can also be displayed in a Dark theme (Dashlane's dark green):

<Playground
  style={{
    display: 'flex',
    flexDirection: 'row',

    margin: '-40px',
    alignItems: 'center'
  }}
>
  <Link theme='dark'>Need help?</Link>
</Playground>
