# Button

A Button that allows users to take actions, and make choices, with a single tap. It can be a either:

- `Contained (filled) button`
- `Text button`

#### Downside: Since Button props extends the default html button props, Docz will display all props including `React.ButtonHTMLAttributes` ones ðŸ˜­

Bug on repo link: https://github.com/doczjs/docz/issues/895

<Props of={Button} />

## Usage

### Default variant for the button is `contained`

<Playground style={{ display: 'flex', flexDirection: 'row', justifyContent: 'space-around' }}>
  <div>
    <Button nature="danger">Danger Button</Button>
  </div>

<div>
  <Button nature='dark'>Dark Button</Button>
</div>

<div>
  <Button nature='light'>Light Button</Button>
</div>

<div>
  <Button nature='secondary'>Secondary Button</Button>
</div>

  <div>
    <Button nature="success">Success Button</Button>
  </div>
</Playground>

### You can have a text button by specifying `variant` property to `text`

<Playground style={{ display: 'flex', flexDirection: 'row', justifyContent: 'space-around' }}>
  <div>
    <Button nature="danger" variant="text">Danger Button</Button>
  </div>

<div>
  <Button nature='dark' variant='text'>
    Dark Button
  </Button>
</div>

<div>
  <Button nature='light' variant='text'>
    Light Button
  </Button>
</div>

<div>
  <Button nature='secondary' variant='text'>
    Secondary Button
  </Button>
</div>

  <div>
    <Button nature="success" variant="text">Success Button</Button>
  </div>
</Playground>

### Buttons (either `contained` or `text`) can be `large` ones

<Playground style={{ display: 'flex', flexDirection: 'row', justifyContent: 'space-around' }}>
<div>
  <Button nature='secondary' variant='text' large={true}>
    Label
  </Button>
</div>

  <div>
    <Button nature="success" large={true}>Label</Button>
  </div>
</Playground>
