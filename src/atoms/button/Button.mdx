# Button

A Button that allows users to take actions, and make choices, with a single tap. It can be a either:

- `Contained (filled) button`
- `Text button`

## Properties

<Props of={Button} />

## Usage

### Default variant for the button is `contained`

<Playground style={{ display: 'flex', flexDirection: 'row', justifyContent: 'space-around' }}>
  <div>
    <Button type="danger">Danger Button</Button>
  </div>

<div>
  <Button type='dark'>Dark Button</Button>
</div>

<div>
  <Button type='light'>Light Button</Button>
</div>

<div>
  <Button type='secondary'>Secondary Button</Button>
</div>

  <div>
    <Button type="success">Success Button</Button>
  </div>
</Playground>

### You can have a text button by specifying `variant` property to `text`

<Playground style={{ display: 'flex', flexDirection: 'row', justifyContent: 'space-around' }}>
  <div>
    <Button type="danger" variant="text">Danger Button</Button>
  </div>

<div>
  <Button type='dark' variant='text'>
    Dark Button
  </Button>
</div>

<div>
  <Button type='light' variant='text'>
    Light Button
  </Button>
</div>

<div>
  <Button type='secondary' variant='text'>
    Secondary Button
  </Button>
</div>

  <div>
    <Button type="success" variant="text">Success Button</Button>
  </div>
</Playground>
