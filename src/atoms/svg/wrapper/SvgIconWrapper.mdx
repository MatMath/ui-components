# SvgIconWrapper

This is a simple button wrapper for your svg icons

## Properties

<Props of={SvgIconWrapper} />

## Usage

### Basic usage

<Playground style={{ display: 'flex', justifyContent: 'center' }}>
  <SvgIconWrapper>
    <CopyPassword />
  </SvgIconWrapper>
</Playground>

### The contained svg can have a bigger size

<Playground
  style={{
    display: 'flex',
    flexDirection: 'row',
    justifyContent: 'space-around'
  }}
>
  <SvgIconWrapper>
    <Edit />
  </SvgIconWrapper>
  <SvgIconWrapper>
    <Edit size={3} />
  </SvgIconWrapper>
</Playground>

### As in the example below, for a disabled behaviour, wrapper should be disabled and icons as well

<Playground style={{ display: 'flex', justifyContent: 'center' }}>
  <SvgIconWrapper disabled>
    <CopyPassword disabled />
  </SvgIconWrapper>
</Playground>
