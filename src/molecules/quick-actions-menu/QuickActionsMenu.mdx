# Quick Actions Menu

Quick action menu is a specific components meant for users to quickly copy
a credential's information like login and password.
It is not meant to be used in an other context. If you're looking for customizable dropdown,
check out the [Dropdown Menu component](./src-atoms-button-button).

## Example

<Playground>
  {() => {
    const [isOpen, setIsOpen] = React.useState(false);
    const onClick = e => {
      e.preventDefault();
      setIsOpen(!isOpen);
    };
    return (
      <div className='playground-style' style={{ height: '200px' }}>
        <QuickActionsMenu
          icon={<MoreIcon />}
          isOpen={isOpen}
          onClick={onClick}
          tooltipTitle='Tooltip'
        >
          <DropdownElement>Copy Password</DropdownElement>
          <DropdownElement>Copy Login</DropdownElement>
        </QuickActionsMenu>
      </div>
    );
  }}
</Playground>
