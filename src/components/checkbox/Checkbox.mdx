# Checkbox

A checkbox component

## Properties

<Props of={Checkbox} />

## Usage

### Basic usage

Examples below show the `Checkbox` in its checked and unchecked states, both in enabled and disabled modes:

<Playground
  className={{preview: 'playground-base'}}
>
  {() => {
      const [isChecked, setIsChecked] = React.useState(false);
  const label = isChecked ? 'Checked' : 'Unchecked';
return (
  <>
    <Checkbox label={label} checked={isChecked} onChange={e => setIsChecked(e.currentTarget.checked)}/>
    <Checkbox disabled />
    <Checkbox disabled checked />
  </>
  )}}

</Playground>

### Custom label

`label` can be a React Component

<Playground
  className={{preview: 'playground-base'}}
>
  {() => {
      const [isChecked, setIsChecked] = React.useState(false);
  const label = (<span style={{color: 'red'}}>Component Label</span>);
return (
<>
<Checkbox label={label} checked={isChecked} onChange={e => setIsChecked(e.currentTarget.checked)}/>
</>
)}}

</Playground>
