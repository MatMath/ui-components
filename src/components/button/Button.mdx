# Button

A Button that allows users to take actions, and make choices, with a single tap.

## Properties

<Props of={Button} />

## Usage

### Nature

<Playground className={{preview: 'playground-base'}}>

    <Button nature='primary'>Primary</Button>
    <Button nature='secondary'>Secondary</Button>
    <Button nature='danger'>Danger</Button>
    <Button nature='ghost'>Ghost</Button>

</Playground>

### Possibilities

All buttons can display icons, text or mixed content.

<Playground className={{preview: 'playground-base'}}>
    <Button nature='primary'>
      <CopyPasswordIcon color={white} style={{ marginRight: '8px'}}/>
      Save
    </Button>

    <Button nature='secondary'>
      <CopyPasswordIcon color={midGreen00} style={{ marginRight: '8px'}}/>
      Cancel
    </Button>

    <Button nature='danger'>
      <CopyPasswordIcon color={white} style={{ marginRight: '8px'}}/>
      Delete
    </Button>

    <Button nature='ghost'>Decline</Button>

    <Button nature='ghost'>
      <CopyPasswordIcon color={midGreen00} />
    </Button>

</Playground>

### Theme

The default theme is `light` (white), but some buttons can be displayed in a `dark` theme (Dashlane's dash green, not an actual _dark_ theme).

<Playground className={{preview: 'playground-base'}} style={{ preview: {backgroundColor: dashGreen00} }}>
    <Button nature='primary' theme='dark'>
      Save
    </Button>

    <Button nature='secondary' theme='dark'>
      Cancel
    </Button>

    <Button nature='ghost' theme='dark'>
      Decline
    </Button>

    <Button nature='ghost' size='small' theme='dark'>
      <CopyPasswordIcon color={white} />
    </Button>

</Playground>

> ⚠️ `danger` nature is not supported in `dark` theme.

### Size

Buttons can be have different sizes [`small`, `medium`, `large`] with `medium` being the default value if none is specified.

<Playground className={{preview: 'playground-base'}} style={{ preview: {minHeight: '150px'} }}>
    <div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', justifyContent: 'space-between' }}>
      <Button nature='primary' size='small'>Save</Button>
      <Button nature='primary' size='medium'>Save</Button>
      <Button nature='primary' size='large'>Save</Button>
    </div>

    <div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', justifyContent: 'space-between' }}>
      <Button nature='secondary' size='small'>Cancel</Button>
      <Button nature='secondary' size='medium'>Cancel</Button>
      <Button nature='secondary' size='large'>Cancel</Button>
    </div>

    <div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', justifyContent: 'space-between' }}>
      <Button nature='danger' size='small'>Delete</Button>
      <Button nature='danger' size='medium'>Delete</Button>
      <Button nature='danger' size='large'>Delete</Button>
    </div>

    <div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', justifyContent: 'space-between' }}>
      <Button nature='ghost' size='small'>Decline</Button>
      <Button nature='ghost' size='medium'>Decline</Button>
      <Button nature='ghost' size='large'>Decline</Button>
    </div>

</Playground>

### Disabled

In disabled pseudo-state, buttons behave as following.

<Playground className={{preview: 'playground-base'}}>

    <Button nature='primary' disabled>Save</Button>
    <Button nature='secondary' disabled>Cancel</Button>
    <Button nature='danger' disabled>Delete</Button>
    <Button nature='ghost' size='medium' disabled>Decline</Button>
    <Button nature='ghost' disabled>
      <CopyPasswordIcon color={midGreen00}/>
    </Button>

</Playground>
