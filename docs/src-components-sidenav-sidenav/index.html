Sidenav

A vertical side navigation panel. Takes all available height.

Properties

Sidenav

collapsed
boolean | undefined

SidenavLink

selected
boolean | undefined
collapsed
boolean | undefined
hasNotification
boolean | undefined
icon
ReactNode
required
label
string
required

Usage

  • Notification, truncated label is too long
  • Selected
  • Notification, truncated label is too long

<div style={{ gridColumn: 1, gridRow: 1 }}>
  <Sidenav>
    <SidenavHeader>
      <BusinessLogo darkMode height={40} width={120} />
    </SidenavHeader>
  </Sidenav>
</div>
<div style={{ gridColumn: 1, gridRow: 2 }}>

  </Sidenav>
</div>
