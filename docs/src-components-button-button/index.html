Button

A Button that allows users to take actions, and make choices, with a single tap.

Properties

nature
"primary" | "secondary" | "danger" | "ghost" | undefined
primary
size
"small" | "medium" | "large" | undefined
medium
theme
"light" | "dark" | undefined
light

Usage

Nature

<Button nature="primary">Primary</Button>
<Button nature="secondary">Secondary</Button>
<Button nature="danger">Danger</Button>
<Button nature="ghost">Ghost</Button>

Possibilities

All buttons can display icons, text or mixed content.

<Button nature="primary">
  <CopyPasswordIcon color={white} style={{ marginRight: '8px' }} />
  Save
</Button>
<Button nature="secondary">
  <CopyPasswordIcon color={midGreen00} style={{ marginRight: '8px' }} />
  Cancel
</Button>
<Button nature="danger">
  <CopyPasswordIcon color={white} style={{ marginRight: '8px' }} />
  Delete
</Button>
<Button nature="ghost">Decline</Button>
<Button nature="ghost">
  <CopyPasswordIcon color={midGreen00} />
</Button>

Theme

The default theme is light (white), but some buttons can be displayed in a dark theme (Dashlane's dash green, not an actual dark theme).

<Button nature="primary" theme="dark">
  Save
</Button>
<Button nature="secondary" theme="dark">
  Cancel
</Button>
<Button nature="ghost" theme="dark">
  Decline
</Button>
<Button nature="ghost" size="small" theme="dark">
  <CopyPasswordIcon color={white} />
</Button>

danger nature is not supported in dark theme.

Size

Buttons can be have different sizes [small, medium, large] with medium being the default value if none is specified.

    display: 'flex',
    flexDirection: 'column',
    alignItems: 'center',
    justifyContent: 'space-between',
  }}

    Save
  </Button>

    Save
  </Button>
  <Button nature="primary" size="large">
    Save
  </Button>
</div>

Disabled

In disabled pseudo-state, buttons behave as following.

<Button nature="primary" disabled>
  Save
</Button>
<Button nature="secondary" disabled>
  Cancel
</Button>
<Button nature="danger" disabled>
  Delete
</Button>
<Button nature="ghost" size="medium" disabled>
  Decline
</Button>
<Button nature="ghost" disabled>
  <CopyPasswordIcon color={midGreen00} />
</Button>
